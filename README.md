# 🌷 Sk. Arif Birthday Surprise Website

## 📁 Folder Structure

```
sk-arif-birthday/
├── index.html       ← Page 1 (flat-lay scene)
├── page2.html       ← Page 2 (birthday surprise scene)
├── style.css        ← Shared styles
├── images/          ← ADD YOUR PHOTOS HERE
│   ├── photo1.jpg   ← Flat-lay photo (Page 1 background)
│   ├── photo2.jpg   ← Beach/birthday photo (Page 2 background)
│   ├── scratch1.jpg ← Scratch card #1 reveal photo
│   ├── scratch2.jpg ← Scratch card #2 reveal photo
│   └── scratch3.jpg ← Scratch card #3 reveal photo
└── audio/           ← ADD YOUR AUDIO HERE
    ├── voice.mp3    ← Voice message audio
    └── song.mp3     ← Birthday song / CD music
```

## 🎮 Interactions

### Page 1 (index.html)
| Element | Action |
|---------|--------|
| ☕ Coffee cup | Click repeatedly → coffee level drops gradually → only cup remains after 12 clicks |
| ✉️ Envelope | Click → beautiful loading screen → navigates to Page 2 |

### Page 2 (page2.html)
| Element | Action |
|---------|--------|
| 📜 Letter card | Click → letter opens with animation → read birthday message |
| 🎴 Scratch cards (×3) | Click → scratch card modal → drag finger/mouse to scratch → photo revealed |
| 🎤 Voice message | Click → plays voice.mp3 (add your file) |
| 💿 CD player | Click → CD disc spins + plays song.mp3 (add your file) |

## 🚀 How to Open

Simply double-click `index.html` to open in your browser.

**No server needed** — this is a pure HTML/CSS/JS website.

## 📸 Adding Your Photos

1. Create an `images/` folder inside `sk-arif-birthday/`
2. Save your flat-lay photo as `photo1.jpg`
3. Save your beach/birthday photo as `photo2.jpg`
4. Save 3 surprise photos as `scratch1.jpg`, `scratch2.jpg`, `scratch3.jpg`

## 🎵 Adding Audio

1. Create an `audio/` folder inside `sk-arif-birthday/`
2. Save your voice message as `voice.mp3`
3. Save your birthday song as `song.mp3`
